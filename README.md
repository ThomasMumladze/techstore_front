# TechStore

Modern e-commerce frontend built with React + TypeScript.

[node-v] -v24.12.0
[npm-start] to start server

# Vite config

server: { open: true, host: true, }

** open:true automatically open server
** host:true available in local host for any devices

# project uses

npm install react-router
npm install react-icons
npm install scss and sass
